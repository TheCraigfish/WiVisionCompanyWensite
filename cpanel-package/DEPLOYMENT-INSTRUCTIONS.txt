### Step 1: Upload Files
1. Login to your cPanel
2. Go to File Manager
3. Navigate to public_html folder
4. Upload ALL files from this package to public_html
5. Extract if uploaded as ZIP

### Step 2: Set Permissions
1. Right-click on send-email.php
2. Change permissions to 755

### Step 3: Configure Email Settings
1. Open send-email.php in a text editor
2. Replace these placeholder values with your actual cPanel email settings:
   - $smtp_host = 'mail.yourdomain.com'
   - $smtp_username = 'contact@yourdomain.com'
   - $smtp_password = 'your_actual_password'
3. Save the file

### Step 4: Test
1. Visit your domain
2. Test contact forms
3. Check that navigation works
